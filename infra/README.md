# infra

Add infrastructure-as-code for your cloud deployment here. Bicep,
Terraform, Pulumi, or your IaC tool of choice — whatever fits your
compliance and ops constraints.

Kept empty in the template so each deployment can pick its own topology.
The framework itself is just two stateless container images plus optional
Cosmos DB / blob storage, so deployment is intentionally
unopinionated.
